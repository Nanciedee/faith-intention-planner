# Faith & Intention Planner 2026-2027

A beautiful, printable planner designed for faith-based intention setting and daily mindfulness tracking.

## View Live Demo
Visit: https://yourusername.github.io/faith-intention-planner

## Features
- Cover Page with elegant watercolor design
- December 2025 Intro & Reflection
- Q1 2026 Focus Page
- Monthly Focus Trackers
- Weekly Rituals Pages
- Daily Micro-Trackers
- Emotional Success Section

## Design Specifications
### Color Palette
- Pastel Blue: #BFD8EB
- Pastel Lilac: #D9C1EB
- Blush: #F6D1D1
- Cream: #FFF5E1

### Typography
- Headers: Georgia (Serif), Bold, 22-36pt
- Body Text: Lato (Sans-serif), Regular, 16-20pt

## Installation
```bash
npm install
npm start
```
